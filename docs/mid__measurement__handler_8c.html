void meas_convert_to_f32_ ( S_Meas_TypeDef * pMeas)

Converts measurement to floating-point format.

Performs conversion using one of two methods:

  1. CALC_METHOD_Q15:
    • Scales Q15 value by high limit
    • value = Q15_to_float(Q15_value * high_limit)
  2. CALC_METHOD_DIRECT:
    • Linear transformation of raw value
    • value = raw * gain + offset
Parameters
[in,out]pMeasPointer to measurement structure
Note
Choose method based on required precision and CPU load

void meas_convert_to_q15_ ( S_Meas_TypeDef * pMeas)

Converts raw ADC value to normalized Q15 format.

Performs conversion based on calculation method:

  • CALC_METHOD_Q15: Applies scaling factor
  • CALC_METHOD_DIRECT: Unity conversion

Conversion formula (Q15): value = (raw >> 1) * scale_factor

Parameters
[in,out]pMeasPointer to measurement structure
Note
Right shift by 1 prevents overflow in multiplication

Initializes a measurement channel with specified parameters.

This function configures a measurement channel with all necessary parameters for signal processing:

  1. Reset all measurement values
  2. Configure scaling and offset compensation
  3. Set up digital filtering
  4. Configure calculation method
  5. Set protection limits
Parameters
[out]pMeasPointer to measurement structure to initialize
[in]offsetCalibration offset in Q15 format
[in]scale_factorScaling factor in Q31 format
[in]alphaLow-pass filter coefficient (0.0 to 1.0)
[in]high_limit_softSoft limit for protection in Q15 format
[in]gain_f32Floating-point gain for direct calculation
[in]offset_f32Floating-point offset for direct calculation
[in]calc_methodCalculation method (CALC_METHOD_DIRECT or CALC_METHOD_Q15)
Note
Alpha = 1/(1 + 2πfT) where f is cutoff frequency and T is sampling period
Warning
Ensure pointer is valid before calling
