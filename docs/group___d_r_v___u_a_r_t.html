Detailed Description

Implementation of Comp driver core functions.

Public APIs and definitions for UART communication.

Implements the initialization and processing functions for UART:

  • DMA-based transmission and reception
  • Buffer management and overflow protection
  • Error detection and handling
  • Non-blocking operation support

Example usage:

drv_uart_init();
drv_uart_transmit((uint8_t*)"Hello", 5);
