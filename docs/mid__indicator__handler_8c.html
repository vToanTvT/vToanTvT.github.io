Configure an individual LED’s operating parameters.

Initializes a single LED indicator with:

  1. GPIO pin mapping
  2. Reset blink counter
  3. Set blinking period

Example:

INDI_ConfigLED(&g_indicator_led1_, GPIO_LED1, MEDIUM);
