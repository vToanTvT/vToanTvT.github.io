Detailed Description

Structure defining a PI controller instance.

Contains all parameters and state variables needed for a PI controller:

  • Configuration parameters (gains, limits)
  • Runtime variables (reference, feedback, error)
  • Internal states (proportional, integral terms)
  • Output and control state
