Detailed Description

Derating lookup table entry.

Defines a point in the derating curve:

  • Threshold: Value at which this coefficient becomes active
  • Coefficient: Derating factor to apply

The coefficient is applied when the measured value is less than or equal to the threshold. Tables are arranged in descending order of thresholds.
