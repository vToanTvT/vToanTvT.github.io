Detailed Description

Handles all fault detection, protection, and recovery mechanisms.

Implements full protection layer with features:

  • Fault detection (OVP/UVP/OCP/SCP/OTP)
  • Debounce filtering
  • Fault mode handling (Latch / Auto-recover / None)
  • Hardware comparator fault integration
  • Safe fault release and reset management

This module implements a complete fault management system for EVSE or power conversion units, including:

  • Over/Under Voltage Protection (OVP/UVP)
  • Over Current Protection (OCP)
  • Short Circuit Protection (SCP)
  • Over Temperature Protection (OTP)
  • Fault debouncing and latching mechanisms
  • Auto-recovery functionality

All measurement signals are handled in Q15 fixed-point format for real-time performance and efficient embedded implementation.
