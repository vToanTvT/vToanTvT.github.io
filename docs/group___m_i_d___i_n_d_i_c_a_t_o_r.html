Detailed Description

Middleware for controlling multiple LEDs with flexible blinking patterns.


This module implements a flexible, non-blocking LED indicator system:

  • Configurable blinking periods (SLOW, MEDIUM, FAST)
  • Independent control for multiple LEDs
  • Timer-based non-blocking operation
  • Runtime reconfiguration of blinking speed

Provides APIs for managing LED indicators with configurable blink rates. Features include:

  • Multiple predefined blink periods (SLOW, MEDIUM, FAST)
  • Independent per-LED control
  • Non-blocking software counter implementation
  • Runtime reconfiguration support

◆ E_Indicator_Blink_Period_TypeDef

enum E_Indicator_Blink_Period_TypeDef

LED blinking period enumeration.

Defines blink speed in sampling time multiples. Example (T_sampling = 1 ms):

  • SLOW → 500 ms (1 Hz)
  • MEDIUM → 200 ms (2.5 Hz)
  • FAST → 100 ms (5 Hz)
Enumerator
SLOW 

Slow blinking: 500 * T_sampling

MEDIUM 

Medium blinking: 200 * T_sampling

FAST 

Fast blinking: 100 * T_sampling

Process LED blinking according to configured timing.

Parameters
[in,out]pledConfigPointer to LED configuration structure.
Note
Should be called periodically (e.g., in main loop or timer ISR).

Process LED blinking according to configured timing.

Implements the blinking logic:

  1. Increment counter
  2. Compare with configured period
  3. Toggle LED and reset counter when period elapsed
Parameters
[in,out]pledConfigPointer to LED configuration structure
Note
Should be called periodically at system sampling rate.
Warning
The system call rate must be at least twice the fastest blinking frequency.
