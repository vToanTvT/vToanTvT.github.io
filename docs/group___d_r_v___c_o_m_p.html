Detailed Description

Implementation of Comp driver core functions.

Public APIs and definitions for comparator driver.

Implements the initialization and processing functions for comparator:

  • Hardware initialization and configuration
  • DAC threshold management
  • Protection state control
  • Fault detection and handling

Example usage:

// Initialize comparator
S_Comp_TypeDef comp_config;
comp_init_(&comp_config);
// Update threshold if needed
comp_config.comp_threshold = new_threshold;
comp_set_threshold_(&comp_config);

int comp_init_ ( S_Comp_TypeDef * pcomp)

Initialize comparator module.

Performs complete initialization sequence:

  1. Parameter validation
  2. Reset protection state
  3. Set default threshold
  4. Configure and start hardware:
    • Program DAC threshold
    • Enable DAC output
    • Start comparator
Parameters
[out]pcompPointer to comparator configuration
Returns
Status code
Return values

-1Invalid parameters (NULL pointer)
Note
