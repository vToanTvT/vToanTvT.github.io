Calibrates the ADC channels.

Executes ADC self-calibration:

  1. Stop ongoing conversions
  2. Calibrate internal references
  3. Adjust channel offsets
  4. Verify calibration results

-2Calibration hardware error
Note
Call during initialization or after temperature change
Warning

Parameters
pADCPointer to ADC data structure
Returns
int Status code
Return values
0Success
-1Conversion error
Note
Non-blocking function, check results using adc_get_result_

Starts an ADC conversion sequence.

Configures and starts continuous ADC sampling:

  1. Validates ADC state
  2. Sets up DMA transfer
  3. Begins conversion sequence
  4. Monitors transfer status
Parameters
[in,out]pADCADC data structure for DMA transfer
Returns
Status code
Return values
0Success
-1Invalid parameters (NULL pointer)
-2Hardware error in DMA setup
Note
Non-blocking operation - check results with adc_get_result_
Warning
Do not modify buffer during active DMA transfer
