Detailed Description

Measurement channel data structure.

Contains all data and parameters for a single measurement channel:

  • Raw and processed values
  • Calibration parameters
  • Signal processing settings
  • Protection limits

Two calculation methods supported (configured via calc_method):

  1. Q15 Fixed-point (CALC_METHOD_Q15):
    • Uses scale_factor and offset for conversion
    • Better performance on MCU
    • Limited dynamic range
  2. Direct floating-point (CALC_METHOD_DIRECT):
    • Uses gain_f32 and offset_f32
    • Higher precision
    • More CPU intensive
