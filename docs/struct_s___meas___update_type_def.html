Detailed Description

Measurement update timing control.

Controls periodic measurement updates:

  • Configurable update period
  • Update counter
  • Update flag for synchronization

Update sequence:

  1. Counter increments each tick
  2. When counter reaches period, flag is set
  3. System processes measurements
  4. Flag is cleared for next cycle
