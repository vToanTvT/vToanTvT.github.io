void LB_system_initialization ( void )

Initialize the entire LadyBug system including all layers.

Initializes the driver layer (ADC, PWM), middleware layer (indicators, measurements, LLC), and application layer (EVSE state machine)

Initialize the entire LadyBug system including all layers.

This function initializes all system components in the following order:

  1. Driver Layer: ADC, PWM, and Fault systems
  2. Middle Layer: Indicators, Measurements, LLC controller, Power control
  3. Application Layer: EVSE state machine
