🌐 Overview

  • The LadyBug Project is a firmware platform developed by EVSELab,
  • designed for high-efficiency and modular EV Charger Control Systems.
  • It provides robust control, monitoring, and protection layers for
  • electric vehicle supply equipment (EVSE), optimized for embedded
  • real-time performance and safety.
