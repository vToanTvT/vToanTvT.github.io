Detailed Description

Implementation of GPIO driver core functions.

Public APIs and definitions for GPIO hardware control.

Implements GPIO pin configuration and control:

  • LED indicator management
  • Relay control for PFC and LLC
  • Pin toggle and readback operations

Example usage:

gpio_set_pin_(&GPIO_LED1);
gpio_toggle_pin_(&GPIO_MCU_LLC_RELAY);
