Detailed Description

Implementation of Comp driver core functions.

Public APIs and definitions for PWM signal generation.

Implements the initialization and processing functions for PWM:

  • LLC converter PWM configuration and control
  • Fan speed PWM generation
  • Dead-time management for complementary outputs
  • Frequency and duty cycle control

Example usage:

pwm_init_();
pwm_enable_output_(&g_pwm_llc_config_);
