Detailed Description

Implementation of ADC driver core functions.

Public APIs and definitions for ADC driver.

This file implements the initialization and processing functions for ADC:

  • Hardware initialization and configuration
  • DMA-based sampling sequence
  • Data conversion and buffer management
  • Calibration and error handling

Example usage:

S_Adc_Data_TypeDef adc_data;
adc_init_(&adc_data);
adc_start_conversion_(&adc_data);
if (adc_get_result_(&adc_data) == 0) {
float vbat = adc_data.ADC_voltage_bat * SCALING_FACTOR;
}
