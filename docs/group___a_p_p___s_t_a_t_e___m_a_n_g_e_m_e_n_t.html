Detailed Description

This file contains the implementation of the EVSE state machine logic, responsible for:

  • Initialization and configuration of states
  • Handling state transitions
  • Fault handling and recovery
Version
1.0.0
Date
2025-10-17
Author
EVSELab

This module implements the main state machine for the EVSE (Electric Vehicle Supply Equipment) system. It is responsible for:

  • State initialization and configuration
  • State transitions and validation
  • Debounce and delay management
  • Fault handling and recovery

Main system states:

  • IDLE: System standby, waiting for vehicle connection
  • PRECHARGE: Preparing DC link before enabling power stage
  • RUN: Normal operation with current control
  • FAULT: Error handling and safe shutdown
Note
The state machine is executed periodically by the main scheduler or interrupt loop.
