Detailed Description

Middleware for controlling LLC resonant converter operation.

This file implements advanced control algorithms for an LLC resonant converter, featuring:

  • Multi-loop control architecture
  • Adaptive mode switching
  • Protection mechanisms
  • Soft-start sequence

All control algorithms use fixed-point Q15/Q31 arithmetic for efficient execution on embedded systems.

Version

Date

Author
EVSELab Co. Jsc.

This module implements a comprehensive control system for an LLC resonant converter with the following features:

  • Dual-mode operation (PFM/PWM)
  • Cascaded voltage and current control loops
  • Soft-start functionality
  • Hybrid control strategy
  • Protection mechanisms

Control hierarchy:

  • Outer loop: Voltage control
  • Inner loop: Current control
  • Mode switching: PFM/PWM based on operating conditions
