Detailed Description

ADC driver for hardware control and data acquisition.

This module provides functionality for:

  • ADC initialization and configuration
  • Multi-channel ADC sampling
  • Battery voltage and current measurement
  • Temperature monitoring for PFC and LLC
  • AC voltage measurement
