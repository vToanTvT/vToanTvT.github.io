Detailed Description

Public APIs and definitions for UART communication.

This file contains all the exported function prototypes, macros, and data structures for the UART driver module. The module provides:

  • Non-blocking UART transmission using DMA
  • Configurable TX/RX buffer management
  • Error detection and handling
  • Buffer overflow protection
