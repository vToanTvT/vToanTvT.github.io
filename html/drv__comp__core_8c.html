Detailed Description

Source file for comparator driver module.

Implements the initialization and processing functions for comparator:

  • Hardware initialization and configuration
  • DAC threshold management
  • Protection state control
  • Fault detection and handling

Example usage:

// Initialize comparator
S_Comp_TypeDef comp_config;
comp_init_(&comp_config);
// Update threshold if needed
comp_config.comp_threshold = new_threshold;
comp_set_threshold_(&comp_config);
