Detailed Description

Common definitions and ramp control utilities for fixed-point operations.

This module provides:

  • Global constants for system timing and fixed-point scaling
  • Type definitions (uq15_t, uq31_t)
  • Utility ramp functions for controlled transitions
  • Saturation and limit protection

All computations are based on fixed-point arithmetic for real-time efficiency.
