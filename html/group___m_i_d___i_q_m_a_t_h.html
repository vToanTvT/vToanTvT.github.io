Detailed Description

Middleware library providing arithmetic operations for Q15/Q31 formats.

This module provides efficient fixed-point arithmetic operations using Q15 and Q31 number formats. Features include:

  • Conversion between float and fixed-point
  • Basic arithmetic operations (+, −, ×)
  • Optimized bit-shift implementations
  • Range checking and saturation

Number formats:

  • Q15: 1 sign bit + 15 fractional bits → range −1.0 to +0.999969
  • Q31: 1 sign bit + 31 fractional bits → range −1.0 to +0.999999999
