Detailed Description

Implementation of LLC resonant converter control system.

Author
ADMIN
Version
1.0.0
Date
2025-10-17
Copyright
Copyright (c) 2025

This module implements advanced control algorithms for an LLC resonant converter, featuring:

  • Multi-loop control architecture
  • Adaptive mode switching
  • Protection mechanisms
  • Soft-start sequence

All control algorithms use fixed-point Q15/Q31 arithmetic for efficient execution on embedded systems.

◆ llc_init_()

void llc_init_ ( void )

Initializes the complete LLC control system.

Initializes LLC converter subsystem.

This function performs a complete initialization of the LLC subsystem:

  1. Sets initial operating mode (PWM)
  2. Configures voltage loop PI controller:
    • Proportional gain: KP_VOL_Q15
    • Integral gain: KI_VOL_Q15
    • Output limits: HIGH/LOW_LIMIT_VOL_Q15
  3. Configures current loop PI controller:
    • Proportional gain: KP_CUR_Q15
    • Integral gain: KI_CUR_Q15
    • Output limits: HIGH/LOW_LIMIT_CUR_Q15
  4. Configures duty cycle PI controller:
    • Proportional gain: DUT_KP_CUR_Q15
    • Integral gain: DUT_KI_CUR_Q15
    • Output limits: DUT_HIGH/LOW_LIMIT_CUR_Q15
Note
All gains are in Q15 fixed-point format

void llc_regulator_current_loop_ ( void )

Executes current regulation loop using frequency control.

Executes current regulation loop.

Implements a single-loop current controller using PFM:

  1. Updates feedback:
    • Reads battery current in Q15 format
  2. Executes PI control:
    • Processes current error
    • Applies anti-windup protection
  3. Updates PWM:
    • Converts output to frequency
    • Updates PWM frequency
    • Sets fixed deadtime

Control flow: 

* Current Feedback → PI Controller → Frequency Command → PWM Update
*         ↑                                                 |
*         └─────────────────── Feedback ──────────────────┘
*
Note
This function implements frequency modulation (PFM) mode for power control

void llc_regulator_voltage_loop_ ( void )

Execute voltage regulation loop for LLC.

Executes cascaded voltage-current control.

  • Reads instantaneous battery voltage and current as feedback.
  • Outer loop: Voltage PI controller generates current reference.
  • Inner loop: Current PI controller regulates the current to the reference.
  • Updates PWM frequency based on inner loop output.
  • Sets fixed deadtime for LLC operation.

◆ llc_standby_()

void llc_standby_ ( void )

Places LLC converter in safe standby mode.

Places LLC converter in standby mode.

This function performs a complete shutdown sequence:

  1. Resets PWM parameters to safe values:
    • Frequency: FRE_LLC
    • Deadtime: DEADTIME_LLC
    • Duty cycle: DUTY_LLC
  2. Disables PWM outputs
  3. Resets all PI controllers:
    • Voltage controller
    • Current controller
    • Duty cycle controller

This sequence ensures:

  • Safe shutdown of power stage
  • No residual energy in resonant tank
  • Clean controller state for next startup
Note
Should be called during fault conditions or normal shutdown

◆ llc_update_integral_()

void llc_update_integral_ ( void )

Updates integral terms of PI controllers.

Updates controller integral terms.

This function adapts the integral terms based on current operating conditions:

Current Controller:

  • Scales integral based on switching frequency
  • Formula: integral = FREQ_CLK / (timebase * FREQ_MAX)

Duty Controller:

  • Scales integral based on duty cycle
  • Formula: integral = current_duty / DUTY_MAX
Note
This dynamic adjustment ensures:
  • Stable control across operating range
  • Proper scaling of control effort
  • Smooth mode transitions
