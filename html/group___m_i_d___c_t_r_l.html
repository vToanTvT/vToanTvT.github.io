Detailed Description

Middleware implementing digital PI control with anti-windup.

This module implements a digital PI controller using fixed-point arithmetic for efficient real-time operation. It supports:

  • Configurable proportional (Kp) and integral (Ki) gains
  • Output limiting and anti-windup protection
  • Enable/disable control state
  • Fixed-point math (q31_t / uq15_t)

Parameters

Prevents integral windup by:

  1. Limiting output between high_limit and low_limit
  2. Back-calculating integral term to match limited output

Applies anti-windup protection to the PI controller.

This function prevents integral windup using the back-calculation method:

  1. If output exceeds limits, it is clamped to the limit
  2. The integral term is recalculated to match the clamped output
  3. This prevents the integral term from growing beyond what can be achieved by the actuator
Parameters
pControllerPointer to PI controller structure
Note
Should be called after controller_process_pi_ but before using the output

Parameters

StatusController state (0:Disabled, 1:Enabled) in uq15_t format
Note
This function should be called periodically at a fixed sampling rate
When disabled (Status = 0), all internal states are reset to zero

Processes the PI controller algorithm.

When enabled, performs the following steps:

  1. Updates controller state
  2. Calculates current error
  3. Computes proportional term
  4. Updates integral term
  5. Combines terms for final output

When disabled, resets all internal states to zero
