Detailed Description

Source file for ADC driver module.

Implements the initialization and processing functions for ADC:

  • Hardware initialization and configuration
  • DMA-based sampling sequence
  • Data conversion and buffer management
  • Calibration and error handling

Example usage:

// Initialize ADC
S_Adc_Data_TypeDef adc_data;
adc_init_(&adc_data);
// Start conversion and get results
adc_start_conversion_(&adc_data);
if (adc_get_result_(&adc_data) == 0) {
// Process ADC data
float battery_voltage = adc_data.ADC_voltage_bat * SCALING_FACTOR;
}
