Detailed Description

Public APIs and definitions for PWM signal generation.

This file contains all the exported function prototypes, macros, and data structures for the PWM driver module. The module provides:

  • LLC converter PWM generation with dead-time control
  • Fan speed control via PWM
  • Multiple channel configuration
  • Frequency and duty cycle management

Initializes PWM modules and configurations.

Performs complete PWM initialization:

  1. LLC Converter PWM Setup
    • Timer base configuration
    • Channel assignment
    • Frequency and duty configuration
    • Dead-time setup
  2. Fan Control PWM Setup
    • Timer base configuration
    • Channel assignment
    • Speed control setup
Note
Must be called before using any other PWM functions
Warning
Ensure clocks enabled before calling

Example usage:

pwm_init_(); // Initialize both PWM channels
if (g_pwm_llc_config_.status_pwm == K_PWM_STATUS_ENABLED_) {

}
