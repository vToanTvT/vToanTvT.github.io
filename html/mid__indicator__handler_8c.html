Detailed Description

Implementation of LED indicator control system.

This module implements a flexible LED indicator system with features:

  • Configurable blinking periods (SLOW, MEDIUM, FAST)
  • Independent control of multiple LEDs
  • Non-blocking timer-based operation
  • Runtime period modification

Configures an individual LED's operating parameters.

This function sets up a single LED configuration by:

  1. Assigning the GPIO pin for control
  2. Resetting the internal blink counter
  3. Setting the desired blink period

Example usage:

// Configure LED1 for medium-speed blinking
INDI_ConfigLED(&g_indicator_led1_, GPIO_LED1, MEDIUM);
