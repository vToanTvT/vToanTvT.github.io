Detailed Description

Middleware for measurement acquisition, scaling, and filtering.

This module defines the interface for the measurement processing system:

  • ADC data acquisition and signal conditioning
  • Fixed-point (Q15) and floating-point calculations
  • Digital filtering with configurable coefficients
  • System measurement channels:
    • Battery current and voltage
    • DC bus voltage
    • AC input voltage
    • PFC and LLC stage temperatures

Supported calculation methods:

  1. Q15 fixed-point: High precision, optimized for MCU.
  2. Direct floating-point: Simpler implementation, higher overhead.
Note
All measurement channels support:
  • Configurable scaling and offset
  • Digital filtering
  • Hard and soft limits
  • Fault detection

Initialize the complete measurement system.

Performs initialization of all measurement components:

  1. Temperature Measurements:
    • PFC stage thermal sensor (direct calculation)
    • LLC stage thermal sensor (direct calculation)
  2. Electrical Measurements:
    • Battery current (Q15 format with filtering)
    • Battery voltage (Q15 format with filtering)
    • DC bus voltage (Q15 format with filtering)
    • AC input voltage (direct calculation)
  3. System Components:
    • Update timing control structure
Note
Each measurement is configured with appropriate:
  • Offset compensation
  • Scaling factors
  • Filter settings
  • Protection limits

void meas_update_all_float_ ( )

Update all measurements to floating-point format.

Converts all channels from native format:

  • Q15 to float conversion
  • Direct calculation updates
  • Final value computation
Note
Called after Q15 updates complete

Update all measurements to floating-point format.

Converts all measurement channels from their native format (Q15 or direct) to floating-point representation:

  1. Battery measurements (current, voltage)
  2. Power stage measurements (DC bus, AC input)
  3. Temperature measurements (PFC, LLC)
Note
Conversion method depends on measurement configuration:
  • Q15: Scales by high limit and converts to float
  • Direct: Applies gain and offset to raw value
